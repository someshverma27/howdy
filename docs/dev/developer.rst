Contribute
==========

RADIS is an open-source project, and therefore anyone can contribute, whether you already
know spectroscopy or not.

You want to become a `contributor <https://github.com/radis/radis/graphs/contributors>`__ ? Welcome !
First step is to read the `Contributing Guide <https://github.com/radis/radis/blob/develop/CONTRIBUTING.md>`__ .

If you want to contribute and have no specific project in mind, have a look at the
`GitHub opened issues <https://github.com/radis/radis/issues>`__.
For a First Time contribution, you can tackle one of the Issues displayed as
`Good First Issue <https://github.com/radis/radis/contribute>`__.

Also, join the community chat, where you'll get a lot of help :

.. image:: https://img.shields.io/badge/slack-join-green.svg?logo=slack
    :target: https://radis.github.io/slack-invite/
    :alt: Slack

.. image:: https://badges.gitter.im/Join%20Chat.svg
    :target: https://gitter.im/radis-radiation/community
    :alt: Gitter

Finally, you can also suggest or vote for new features below:

.. image:: https://feathub.com/radis/radis?format=svg
   :target: https://feathub.com/radis/radis


Sources
=======

To modify RADIS, you will need to clone the Git repository.
See the :ref:`detailed installation procedure <label_install>` below.

`Good First Issue <https://github.com/radis/radis/contribute>`__.
Learn more in the :ref:`Contribute <label_developer_contribute>` section.
